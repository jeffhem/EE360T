# Visualizing Runtime Test Coverage

A web-based visualizer for runtime test coverage.

### Installation

```
npm install
```

### Start Dev Server

```
npm run dev
```
Server hosting at `localhost:8080' with watch and auto-reload

### Build Prod Version

```
npm run build
```
Built files are output in the `dist/` folder

### Usage

The application is looking for json file `results.json` in the root directory to initiate the graph generation.
